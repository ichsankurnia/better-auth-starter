# Better-Auth, Drizzel ORM, @neondatabase/serverless

## Better Auth Generate Table and Schema

- Follow instruction from this link [Better Auth Installation](https://www.better-auth.com/docs/installation) and this [Todo App with Neon Postgres](https://orm.drizzle.team/docs/tutorials/drizzle-nextjs-neon)

### Generate schema/table using better-auth cli, not using drizzle-kit generate

> npx @better-auth/cli generate

- Copy all schema in auth-schema.ts to /db/schema.ts

### Push changes directly to the database using

> npx drizzle-kit push

- delete file auth-schema.ts
